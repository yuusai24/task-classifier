# ALC キャンプ 成果まとめ（友彩）

## 作ったもの

Claude Code で**講座アーカイブの自動生成フロー**を構築した。

- `archive_summary_prompt.txt`：文字起こしを貼るだけで HTML 整形するプロンプト
- `archive_template_utage.html`：UTAGE（会員サイトプラットフォーム）用 HTML テンプレート（4セクション）
- 配色を 2 回調整（マゼンタ・ライムグリーン・水色）
- 「金運爆上げ強化月間」Day2〜Day4 のアーカイブ HTML を 3 本生成・コミット済み

Web セッションから GitHub に直接コミット・プッシュする流れが動いた。

## 詰まったところ

- **差し替え箇所が不明瞭**：どこを書き換えるかわからず、コメント明示版に作り直した（解決）
- **文字起こしの精度ブレ**：元音声の質で要約がばらつく。目視確認で対処。未解決
- **リポジトリ名のズレ**：`task-classifier` で立てたが実際はアーカイブ生成。実害なし
